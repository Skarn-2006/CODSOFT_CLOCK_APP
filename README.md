# Buzzify Landing Page

A beautiful and modern landing page for Buzzify, the ultimate clock app.

## Features
- **Responsive Design:** Optimized for different screen sizes.
- **Dynamic Clock:** Shows real-time clock preview in a phone mockup.
- **Micro-animations:** Elegant interactions and animated blob background effects.
- **Direct Download:** Integrated download link for the Buzzify APK.

## Tech Stack
- HTML5
- Vanilla CSS3
- Vanilla JavaScript

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your web browser.

Alternatively, you can serve the directory using a local web server (e.g., `npx serve .` or VS Code Live Server).
